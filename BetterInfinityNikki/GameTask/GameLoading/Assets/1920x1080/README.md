# 图像资源说明

## 需要准备的图片

### 1. enter_game.png ✅
- **描述**："进入游戏"按钮的截图
- **分辨率**：1920x1080（或其他游戏运行分辨率）
- **要求**：
  - 只截取按钮核心区域
  - 确保图片清晰
  - 保存为 PNG 格式
  - 文件名：`enter_game.png`
- **识别区域**：屏幕下半部分中间 2/3 区域（X: 1/6~5/6, Y: 1/2~1）

### 2. meiyali_menu.png ✅
- **描述**："美鸭梨"菜单按钮截图（主界面标志）
- **分辨率**：1920x1080（或其他游戏运行分辨率）
- **要求**：
  - 截取主界面左侧的"美鸭梨"菜单按钮
  - 确保图片清晰
  - 保存为 PNG 格式
  - 文件名：`meiyali_menu.png`
- **识别区域**：屏幕左上角 1/5×1/5 正方形区域（X: 0~1/5, Y: 0~1/5）

## 如何获取图片

1. 启动无限暖暖游戏
2. 等待到登录/启动界面
3. 使用截图工具截取"进入游戏"按钮
4. 裁剪到合适大小（建议 200x60 像素左右）
5. 保存到本目录

**对于美鸭梨菜单按钮**：
1. 进入游戏主界面
2. 找到左侧的"美鸭梨"菜单按钮
3. 截图并裁剪（建议 100x100 像素左右）
4. 保存到本目录

## 注意事项

- 图片质量会影响识别准确率
- 如果游戏有多个语言版本，可能需要准备多个图片
- 建议使用默认语言（中文）的截图
- 美鸭梨菜单按钮是判断是否进入主界面的重要标志
