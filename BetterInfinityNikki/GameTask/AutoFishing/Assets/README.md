# 自动钓鱼资源文件说明

## 文件夹结构

需要在以下位置放置模板图片：

```
BetterInfinityNikki/GameTask/AutoFishing/Assets/
├── 1920x1080/              # 1080p分辨率的模板图片
│   ├── cancel_fishing.png  # 取消钓鱼按钮
│   ├── raise_rod.png       # 提竿按钮（鱼上钩）
│   ├── pull_fishing_line.png # 拉扯鱼线提竿按钮
│   ├── reel_line.png       # 收线按钮
│   └── skip_animo.png      # 跳过动画按钮
└── [其他分辨率]/            # 其他分辨率的模板图片（可选）
```

## 模板图片说明

### 1. cancel_fishing.png
- **用途**: 检测是否进入钓鱼状态
- **出现时机**: 手动抛竿后，整个钓鱼过程中都存在
- **位置**: 屏幕右下角
- **检测范围**: 宽30%，高20%

### 2. raise_rod.png
- **用途**: 检测鱼是否上钩
- **出现时机**: 鱼上钩后，未提竿前存在
- **位置**: 屏幕右下角
- **检测范围**: 宽35%，高20%

### 3. pull_fishing_line.png
- **用途**: 检测是否进入拉扯鱼线阶段
- **出现时机**: 提竿后进入拉扯阶段时存在
- **位置**: 屏幕右下角
- **检测范围**: 宽35%，高20%

### 4. reel_line.png
- **用途**: 检测是否进入收线阶段
- **出现时机**: 拉扯结束后进入收线阶段
- **位置**: 屏幕右下角
- **检测范围**: 宽35%，高20%

### 5. skip_animo.png
- **用途**: 检测钓鱼成功后的结算动画
- **出现时机**: 钓鱼成功后播放结算动画时
- **位置**: 屏幕右下角
- **检测范围**: 宽20%，高20%

## 如何获取模板图片

### 方法1：截图工具
1. 在游戏中进入对应的钓鱼阶段
2. 使用截图工具截取右下角的按钮区域
3. 保存为PNG格式
4. 放入对应的分辨率文件夹

### 方法2：从BGI复制
如果无限暖暖的UI与原神相似，可以尝试从BGI的资源文件夹复制：
```
BetterGenshinImpact/GameTask/AutoFishing/Assets/1920x1080/
```

### 注意事项
- 图片格式必须为PNG
- 建议截取清晰的按钮图标部分
- 背景尽量干净，避免多余元素
- 不同分辨率需要分别准备对应的图片

## 当前实现的功能

✅ **状态机流程**：
1. Idle → Fishing（检测到取消钓鱼按钮）
2. Fishing → FishBite（检测到提竿按钮）
3. FishBite → PullingLine（自动提竿后检测到拉扯按钮）
4. PullingLine → Reeling（拉扯完成检测到收线按钮）
5. Reeling → Success（收线完成检测到跳过按钮）
6. Reeling → PullingLine（继续钓下一条鱼）
7. Success → Fishing/Idle（跳过动画后）

✅ **自动操作**：
- 自动提竿（左键点击）
- 自动拉扯（智能检测黄色进度条，A/D键控制）
- 自动收线（右键连续点击）
- 自动跳过动画（F键）

✅ **智能拉扯策略**：
- 检测屏幕中心60%×40%区域
- 识别黄色圆形饼图 RGB(255, 220, 165) - RGB(230, 250, 200)
- 统计左右两侧黄色像素分布
- 黄色多的一侧表示进度低，向该方向拉扯
- 如果无法判断，使用左右交替降级策略

## 测试步骤

1. 准备所有模板图片并放入正确位置
2. 在TriggerSettingsPage中启用"半自动钓鱼"
3. 手动抛竿进入钓鱼状态
4. 观察遮罩层"钓鱼"状态变为绿色
5. 等待鱼上钩，观察自动提竿
6. 观察自动拉扯和收线过程
7. 观察自动跳过动画

## 调试技巧

所有识别对象都设置了 `DrawOnWindow = true`，会在屏幕上显示识别框，方便调试。

可以在日志中看到详细的状态转换信息：
- 🎣 检测到钓鱼界面
- 🐟 鱼上钩了
- ⬆️ 自动提竿
- 🎯 进入拉扯鱼线阶段
- ✅ 拉扯完成，进入收线阶段
- 🎉 钓鱼成功
- ⏭️ 跳过结算动画
